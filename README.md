# THE QURAN PROJECT(Name TBA)

### 概要：
- to make a quran app
- that is easy to use, focused towards to people who are doing or have done hifz.
- also aims to provide the best available translation, in english and japanese (with furigana)
- So primarily my use case. if it fits others good for them.

### 機能
- the ability to track ones reading progress and the the health of a particular page or ayat. the longer the last time of revision the worse the health.
	-> this will help users to figure out which parts of the quran they are neglecting
	-> should show the overall health of the quran and its well being.
- allow the user to be as atomic as possible. ayath, ruku, page, surah or juzu. thus giving a birdseye view of their state.
- allows the user to track their mistakes. in each ayath.
	- by allowing them to select or write down the type of mistake that they are facing
	- select the specific area within the ayath if required to figure out what the mistake is.
	- give each type of mistake a color. and make them visible through color when the user is reciting
- add like a ramadhan phase or round.
- japanese: toggle furigana on off
- Link the mutashabihath.
- I want it to be a tool to learn arabic as well. but maybe i am adding too much to it in one shot. Can be considered in the long run.
